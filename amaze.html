Amaze

Amaze is a game I designed for the 2014 UC Berkeley UPE Game Jam, a 24 hour event, with my teammate, Anton Holmberg. Anton already had an idea for a game, so we identified the constraints (time, skills, and the most appropriate platform and aesthetic for that type of game). I designed the interface and graphics and Anton coded the game in Xcode using Objective C. Amaze won third place in the game jam!
