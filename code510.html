Code 510

Problem

Local youth can benefit from mentorship for their self-driven tech projects and opportunities to explore computing careers.

Action

Support game development in weekly meetings with students. Design and build a website to host cirriculum and outreach.

Result

My mentees have successfully designed and built 1 game. Students are using the website to take intro classes.

Overview

Code 510 is an after school program where youth from the East bay can find mentorship for their self driven projects and learn computing skills. I have volunteered as a mentor for Code 510 for the last 2 years and have supported two game projects: one in javascript and one in 3D using Unity and Autodesk Maya. I advise students in the areas of design, concept art, Maya, and code.

Recently I designed and built the website to host Code 510’s curriculum and outreach using Wordpress.
